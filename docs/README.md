## 是什么

Valine 是一款基于[Leancloud](http://leancloud.cn)的无后端评论系统，简单、易用、快速、安全。

查看[快速开始](quickstart.md)了解详情。

## 特性
- 快速
- 安全
- 无后端实现
- MarkDown 全语法支持
- 轻量易用(~17kb gzipped)
